# Analisis de Ventas de Videojuegos

## Descripcion General

Este proyecto consiste en un Analisis Exploratorio de Datos (EDA) sobre ventas historicas de videojuegos.

El objetivo es identificar patrones y tendencias relacionadas con generos, plataformas, publishers y regiones, utilizando herramientas de analisis de datos en Python.

## Objetivos

* Identificar los generos con mayores ventas.
* Analizar el rendimiento de distintas plataformas.
* Comparar las ventas entre regiones.
* Estudiar la evolucion de las ventas a lo largo del tiempo.
* Explorar relaciones entre ventas regionales y ventas globales.

## Herramientas Utilizadas

* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset

El conjunto de datos contiene informacion historica sobre ventas de videojuegos, incluyendo:

* Nombre del juego
* Plataforma
* Genero
* Publisher
* Año de lanzamiento
* Ventas por region
* Ventas globales

## Proceso de Analisis

### Limpieza de Datos

* Revision de la calidad de los datos.
* Eliminacion de columnas no relevantes para el analisis.
* Verificacion de valores faltantes.
* Seleccion de variables de interes.

### Analisis Exploratorio

Durante el proyecto se analizaron:

* Ventas por genero.
* Ventas por plataforma.
* Ventas por publisher.
* Diferencias entre regiones.
* Evolucion temporal de las ventas.
* Correlaciones entre mercados.

## Principales Hallazgos

* Action fue el genero con mayores ventas globales.
* Norteamerica concentro el mayor volumen de ventas.
* Existen diferencias significativas entre las preferencias de cada region.
* Algunas plataformas acumularon gran parte de las ventas del mercado.
* Las ventas regionales presentan una relacion importante con las ventas globales.
